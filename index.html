About

Hey there! My name is Elijah Luis, and I work as a Web Designer / Front-End web developer in the Philippines. I get up in the morning, make a cup of coffee, turn on my computer, and start creating things that are constantly accessible online.

I'm particularly good at bringing design ideas to life on the Front-End. I can create engaging and responsive user interfaces thanks to my proficiency in HTML, CSS, and JavaScript. I pay close attention to every last detail to deliver an outstanding user experience, whether it's designing gorgeous layouts, enhancing performance, or making sure that everything is compatible across all browsers.

Along with my technical abilities, I am an expert user of version control programs like Git / Bash / GitHub Desktop , facilitating efficient code management and easy communication. I am skilled at applying cutting-edge development approaches like Agile or Scrum, which enables me to quickly react to shifting project needs and produce outcomes on schedule.
